Add this to OBS

  1. Copy your overlay URL with the Copy button above.
  2. In OBS, under Sources click + and choose Browser. Name it (e.g. "Raid Train").
  3. Paste your URL into the URL field.
  4. Set Width 1920 and Height 1080 (a standard 1080p canvas).
  5. Leave the background transparent — the overlay is already see-through, so don't add a color.
  6. Click OK. Your train appears.
  7. Leave the source at full size, positioned at 0, 0 — don't drag or scale it in OBS. Set the train's Train size and Vertical position (both under Look) with the sliders here instead. They bake into the overlay and travel with the URL across scenes and machines, and the train always stays fully on screen.

Changed something? Copy the new URL and paste it back into the same Browser source's URL field.

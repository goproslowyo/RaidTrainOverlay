A built-in demo lineup — no event needed. Build yours with your own slug.

How it works

  1. Open the Configurator. Paste your event's RaidPal link — the whole URL is fine, it grabs the event for you — then pick a theme and options and copy the overlay URL.
  2. Add it to OBS. In OBS: Sources → + → Browser, paste the URL, and size the source to your canvas. The overlay background is transparent.
